# The Egyptian History Portal
[![Ask DeepWiki](https://devin.ai/assets/askdeepwiki.png)](https://deepwiki.com/OmarDev709/The_Egyptian_History_Portal.git)

The Egyptian History Portal is an immersive web experience designed to take you on a journey through the rich history of Ancient Egypt. Created for the Egyptian State Award for Young Innovator competition, this project serves as a visually engaging and interactive educational tool.

## Features

*   **Interactive Timeline:** Explore major historical periods from the Pre-Dynastic Era to the Greco-Roman conquest, complete with key events, images, and descriptions.
*   **Virtual Museum Guides:** Discover two of Egypt's most famous museums:
    *   **The Egyptian Museum (Tahrir):** View galleries, floor plans, visitor information, and its location on the map.
    *   **The Grand Egyptian Museum (GEM):** Get a glimpse into the world's largest archaeological museum, featuring photo galleries, maps, and visitor details.
*   **Dual-Theme UI:** Choose between an atmospheric dark mode and a high-contrast light mode for comfortable viewing.
*   **Engaging Visuals:** Features an animated desert landscape with pyramids and stars, a game-like navigation interface, and smooth modal transitions for a dynamic user experience.
*   **Fully Responsive:** The portal is designed to be accessible and functional on desktops, tablets, and mobile devices.
*   **Performance Optimized:** Utilizes lazy loading for images to ensure fast load times.

## Portal Sections

The portal is organized into several key sections accessible via the main navigation bar:

*   **Home:** The main entry point, featuring an overview of Egyptian civilization and a detailed historical timeline.
*   **The Egyptian Museum:** A deep dive into the classic museum in Tahrir Square, showcasing its famous artifacts and providing practical information for visitors.
*   **The Grand Egyptian Museum:** A modern showcase of the new GEM, highlighting its architecture, key exhibits like the treasures of Tutankhamun, and visitor information.
*   **Credits:** Acknowledgment of the developer, contact information, and a list of the technologies used to build the project.

## Technologies Used

*   **HTML5:** For the structure and content of the web pages.
*   **CSS3:** For all styling, layout, animations, and the responsive design.
*   **JavaScript:** For interactivity, navigation, theme switching, and dynamic content animations.

## How to Use

To run the portal locally:
1.  Clone the repository:
    ```bash
    git clone https://github.com/OmarDev709/The_Egyptian_History_Portal.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd The_Egyptian_History_Portal.git
    ```
3.  Open the `index.html` file in your preferred web browser.
